# OBJ-20260221-002 Payload

Objective: Exercise blocked/recovered and reopen-rate behavior in QSOP metrics.

Scenario design:
1. Objective is accepted.
2. It becomes blocked.
3. It recovers to in-progress.
4. It reaches completed.
5. It re-enters blocked/in-progress after completion.
6. It completes again.

Expected metric effects:
- Objective remains completed at end-state.
- Reopen detection increments for this objective.
- Validation remains clean with linked objective/ack chain.
