# Objective Payload: OBJ-20260306-001

## Goal

Finalize the standalone `phivm` runner to load and execute the `.phivm` bytecode natively without the compiler toolchain.

## Scope (Allowed)

- `src/bin/phivm.rs`
- `src/phi_ir/vm.rs`
- Evaluation logic required to bootstrap bytecode execution directly.

## Scope (Do Not Touch)

- Parser, Lowering, Optimizer logic natively (they belong to `phiflow` executable).
- The `WasmHostBridge`.

## Acceptance Tests

1. `cargo run --bin phiflow -- examples/simple.phi -o examples/simple.phivm`
   Expected: Generates `.phivm` containing correct bytes.
2. `cargo run --bin phivm -- examples/simple.phivm`
   Expected: Executes the bytecode identically to evaluator output.

## Handoff Requirements

- Ack state target: `completed`
- Required evidence lines:
  - VM Result prints correct numerical output matching Evaluator.
- Required files_touched format: workspace-relative paths in ACK.

## Notes

- Phase 7 Epoch candidate: PhiVM Runtime must run autonomously. The interpreter handles PhiFlow syntax, the `phivm` runner handles the bytecode structure.
